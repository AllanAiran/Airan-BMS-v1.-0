# Airan Battle Management System (ABMS)

## Overview

Airan Battle Management System (ABMS) is a prototype web-based command and control platform designed for research and educational purposes.

## Features

- Interactive map
- Vehicle tracking (simulated)
- Real-time updates
- Vehicle status dashboard
- Messaging
- Mission planning

## Tech Stack

Frontend
- React
- TypeScript
- Leaflet

Backend
- FastAPI
- Python

Database
- PostgreSQL
